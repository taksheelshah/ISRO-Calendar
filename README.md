# Calendar Stream for ISRO launches
[![Update Calendar](https://github.com/taksheelshah/ISRO-Calendar/actions/workflows/actions.yml/badge.svg)](https://github.com/taksheelshah/ISRO-Calendar/actions/workflows/actions.yml)

An automated calendar feed for upcoming ISRO (Indian Space Research Organisation) launches that uses Launch Library 2 by [thespacedevs](https://thespacedevs.com) to generate a `.ics` stream for ISRO launches

**Stream URL:** `https://taksheelshah.github.io/ISRO-Calendar/isro_launches.ics`

## How to use
### Google Calendar
- Click the `+` next to other calendars
- Choose **From URL**
- Paste the Stream URL
### Outlook
- Add Calendar
- Subscribe from web
- Paste the Stream URL
### Apple Calendar
- File -> New Calendar Subscription
- Paste the Stream URL

Deployed using github pages and updated using github actions, written in python
